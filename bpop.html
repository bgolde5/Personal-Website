About

This program is called "Bubble Pop". It's simliar to candycrush but much more rudimentary. Click the image if you want to see my code.

Main Prerequisits

Knowledge of C and depth first search.

Description

The game is played by selecting a "balloon". If the balloon selected has a similar pair adjacent to it, both balloons are popped, the score is updated, and all balloons below float to fill the space of the popped balloons.

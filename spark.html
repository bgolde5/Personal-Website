About

Spark is a club that I've co-founded with some other motivated computer science students. I'm currently serving as the president.

Description

We started Spark as way to create the space for computer science students to learn new programming skills, network with others, make new friends, further develop their leadership abilities, have fun, and grow! We primarily focus on group programming projects to gain experience working in team environments.
